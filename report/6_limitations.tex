\section{Limitations}
\begin{itemize}
  \item \textbf{LLM Latency Overhead:} The sequential multi-call pipeline introduces a cumulative latency, acceptable here but unfeasible for real-time, millisecond-range environments.
  \item \textbf{Lack of Competitive Evaluation:} Benchmarks were run without active adversarial agents, so the resilience of the utility functions under dynamic opponents remains untested.
  \item \textbf{Statistical Significance:} Results come from a single run per map; multiple runs would be needed to account for environmental stochasticity.
\end{itemize}
