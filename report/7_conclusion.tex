\section{Conclusion}\label{sec:conclusion}

This work presented a deliberative autonomous agent for the Deliveroo.js environment, combining a BDI loop, a PDDL-based path planner, an LLM natural-language interface, and a cooperative multi-agent protocol.

The PDDL planner outperformed the BFS baseline on average, confirming that the overhead of optimal path computation pays off on structurally complex maps, while the BDI loop remained a reliable reactive backbone overall. In the multi-agent scenario the picture is more nuanced: the LLM-powered agent outperforms its pure-BDI counterpart on several maps, showing that natural-language reasoning can translate into actionable decisions, but on maps with faster parcel turnover the cumulative LLM latency erodes this advantage and the reactive agent scores higher.

Taken together, the results confirm that combining symbolic reasoning with natural-language understanding produces a more versatile agent, capable of handling instructions that a purely reactive system could not interpret. The main trade-off is speed: richer reasoning takes more time, and fast-paced environments tend to reward quick reactions over deliberation. Closing this gap remains an open direction for future work.
